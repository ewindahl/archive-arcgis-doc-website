System Requirements

Collector requires certain device requirements to run, as well as having requirements on the data that is used within the application. Collector is supported in a number of languages.

Data requirements

To use your data in Collector, you must use ArcGIS.com or Portal for ArcGIS version 2.1 or later.

Device requirements

Android

  • Android 4.0 (Ice Cream Sandwich)
    or higher operating system
  • Armv7 processor
  • OpenGL ES 2.0 support

iPhone

  • iOS 5 or higher

Supported languages

Arabic, Danish, Dutch, English, Estonian (Android only), French, German, Hebrew, Italian, Japanese, Korean, Latvian (Android only), Lithuanian (Android only), Norwegian, Polish, Portuguese (Brazil), Portuguese (Portugal), Romanian, Russian, Simplified Chinese, Spanish, and Swedish.

The language used by the application is determined by your phone's language setting.

What's New

Collector for ArcGIS version 10.1.12 was released in March 2013. It includes the following enhancements:

  • Localized into additional languages
  • Various bug fixes

Previous version

Version 10.1.11 was released in early March 2013, including the following enhancements:

  • Open maps by signing in using your ArcGIS Online account.
  • In-app search for your maps—When browsing your available maps, you can now search them for specific maps.
  • More sample maps—Tap the Try it now option from the app's initial screen, or see Try it now for information about accessing the sample maps.
