Create Standard Reports

Standard reports allow you to create reports for your sites. You can choose the report types, view report descriptions, choose the report format (PDF, Excel or both) and run favorite reports. Popular reports are indicated based on the frequency of selection. You can create all-in-one PDF reports to email or share with others in your organization, groups or shared with everyone. You can also run individual reports.

Note:

Reports are not available for all countries.

  1. Click the Reports tab and click Standard Reports.

    Your saved sites appear in the site panel.

  2. Choose which sites to include by doing one of the following:
    • Click Select All to automatically select all sites.
    • Check the check box for the sites for which you want to run reports.
    Tip:

    You can click None to uncheck all sites then proceed to check the individual sites for which you want to run reports.

    The number of sites checked appears next to Selected Sites.

Run Standard Reports

You can select the type of reports that you want to view.

  1. Click the All Reports drop-down menu to select the type.

    You can choose from All Reports, Business, Consumer Spending, Demographics, Maps, Market Potential, Tapestry Segmentation, and Traffic. All Reports is the default.

    Once you select from the list, the reports in the category appear.

    2. You can also use the Search for reports text box to find reports.
  2. Click Add to add the report to the Selected Reports table.

    Using the drop-down menu, you can choose the format of the report. You can choose from PDF, Excel or both.

    The Sample column provides a sample view of the report. Click to view a sample report.

    The yellow star indicates a favorite report. Click the star to select favorite reports. When selected, the star will turn yellow.

    The report is added to the Selected Reports table for the selected sites. Continue this process until you have selected all the reports that you want to run.

    To remove the report from Selected Reports, click the X in the red box.

  3. You can enter a subtitle for the report.
  4. The Generate an all-in-one PDF report and send notification to check box creates an all-in-one PDF report and emails it to you. The check box default setting can be made in Preferences. You can also send the report to others using the CC text box populated with the email addresses that you've set in Preferences. You can change the email addresses here for just this report but it will default to the Preference settings when running future reports. Use only a semicolon to separate multiple email addresses. Click Run all.

    An email is sent with the url to the all-in-one PDF report but is only available for a limited time. The all-in-one PDF report is also available in My Content > My Reports > Ordered Reports. If you've shared on ArcGIS Online, your reports can also be found in ArcGIS Online > My Content > BAO > My Reports > All In One Reports.

    A few things to remember when running reports:
    • All-in-one PDF reports are only created for PDFs. If you create a report with both Excel and PDF reports, the Excel files will not be included.
    • Using map reports when creating all-in-one PDF reports results in large PDF reports that take more time to run and download.
    • If Generate an all-in-one PDF report and send notifications to is disabled in Standard Reports then the report size limit has been exceeded. There is a limit of 10 site map reports (that include site details map, site map and site map on satellite imagery) and a total report limit of 60. You will need to delete reports that exceed the limit to run an all-in-one PDF report.
    • When you click Run all, an all-in-one PDF report is created with the selected reports. If you don't want a report included in the all-in-one PDF report, click the X in the red box to delete it.
    • Only 20 reports can be processed at one time. If you have more than 20 reports, click Run all after each email. The next 20 reports will run and create a new all-in-one PDF report. Continue this for more reports but remember there is a 60 report limit. The last email will contain all reports in the all-in-one PDF report.
    • All-in-one PDF reports canâ€™t exceed the 50 MB limit for sharing on ArcGIS Online.
  5. In the Your All-In-One PDF Report is Ready dialog box, use the options to share the report you've created. Check the Share your all-in-one PDF report on ArcGIS Online check box to enable the sharing functionality.
    Note:

    There is a 50 MB file size limit for sharing on ArcGIS Online. If your all-in-one PDF report exceeds the limit, you can't share.

  6. You can change the default title, add a description and add tags for searching. You can share with your groups, organizations and everyone. Your options for sharing may vary depending on your organizations security and roles. See your administrator to change your permissions.
    Note:

    Sharing with everyone makes your report public. Anyone, including anonymous users, can access and see what you've shared.

  7. Click Share.
  8. You can run reports individually.
    • To run each report individually, click Run now. When the report is ready, the Run now text is replaced with Open Report. Click Open Report to view.

Select favorite reports

To add a report to favorites, click the star to the left of the report name.

When selected, the star turns yellow.

To remove the report from your favorites, click the yellow star to remove the yellow fill.

Run Favorite reports

If you want to run a report for your favorites, use the Run Favorites button.

  1. Click Run Favorites in the Run Standard Reports pane.
  2. Click Open Report.

    Some reports are not available for a polygonal site. If you've selected one of these reports, an error message appears. Check the list below for reports that are not available.

    Reports not available for polygonal sites are:

    • Business Locator
    • Traffic Count Profile
    • Major Shopping Center Locator
