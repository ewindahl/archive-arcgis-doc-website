要件およびダウンロード

Workforce をオフィスで Web アプリと一緒に使用するときや、現場でモバイル アプリと一緒に使用するときは、ArcGIS の組織アカウントがあり、レベル 2 のメンバーである必要があります。 Workforce プロジェクトを作成するには、プロジェクト所有者が組織の管理者または公開者である必要があります。 ArcGIS Online および ArcGIS Enterprise 10.5 以降がサポートされています。

Web アプリを開く

Web アプリ

デスクトップ

  • Chrome 54.0 以降
  • Firefox 49.0 以降
  • Internet Explorer 11.0 以降
  • Edge 37.0 以降
  • Safari 9.0 以降

タブレットとスマートフォン

  • Android 4.0 以降での Chrome 54.0 以降
  • iOS 6.0 以降での Safari

ボスニア語、簡体字中国語、繁体字中国語 (台湾)、繁体字中国語 (香港)、クロアチア語、チェコ語、デンマーク語、オランダ語、英語、エストニア語、フィンランド語、フランス語、ドイツ語、ギリシア語、インドネシア語、イタリア語、日本語、韓国語、ラトビア語、リトアニア語、ノルウェー語、ポーランド語、ポルトガル (ブラジル) 語、ポルトガル (ポルトガル) 語、ルーマニア語、ロシア語、セルビア語、スペイン語、スウェーデン語、タイ語、トルコ語、ベトナム語

モバイル アプリ

Android

  • Android 4.2 (Jelly Bean) 以降
  • ARMv7 または x86 プロセッサ
  • OpenGL ES 2.0 のサポート

iOS アプリ

  • iOS 8 以降
  • iPhone および iPad

サポートされている言語

アラビア語、ボスニア語*、簡体字中国語、繁体字中国語 (台湾)、繁体字中国語 (香港)、クロアチア語、チェコ語、デンマーク語、オランダ語、英語、エストニア語*、フィンランド語、フランス語、ドイツ語、ギリシア語、ヘブライ語、インドネシア語、イタリア語、日本語、韓国語、ラトビア語*、リトアニア語*、ノルウェー語、ポーランド語、ポルトガル (ブラジル) 語、ポルトガル (ポルトガル) 語、ルーマニア語、ロシア語、セルビア語*、スペイン語、スウェーデン語、タイ語、トルコ語、ベトナム語

*Android のみ
