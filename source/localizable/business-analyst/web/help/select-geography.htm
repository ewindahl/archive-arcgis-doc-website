Select Geography

Select Geography allows you to select geographies using Search, Select from map and Select from full list.

To access Select Geography, click Maps, Define Areas for Reports and click Select Geography.

Search

Search allows you to enter the city, county, ZIP Code, block groups, census tracts and so on. You can also enter part of a name or number to search.

Note:

Search will return the first 100 results.

  1. In the Select Geography dialog box that opens on the left, click Search.
  2. In Search for your geography, enter information such as the city, county, ZIP Code or area name.
  3. Click the magnifying glass.

    The search results appear.

    When you check the check box for any of the search results, they appear on the map. Your selection is added to Selected Items.

  4. Click Next.
  5. Your site has been saved. To finish, click Done.

    Now that you've saved your site, you can create standard reports and comparison reports using Get Reports and Organize your sites.

Select from Map

Select from Map allows you to select directly from the map.

  1. In the Select Geography dialog box that opens on the left, click Select from Map.
  2. Click on a geography from the list.
  3. The selected geography boundary and labels appear on the map. Click on the map to select them.

    Once selected, the geographies are combined into one site. This is done by default. Click the No check box if you don't want to combine sites.

  4. Once you've made your selections, click Next.

    The results appear on the map.

  5. Click Done.

    Now that you've saved your site, you can create standard reports and comparison reports using Get Reports and Organize your sites.

Select from Full List

Select from Full List provides a list of geographies regardless of the current map view.

  1. In the Select Geography dialog box that opens on the left, click Select from Full List.
  2. Click on the type of geography from the list.
  3. The selected geography appears with options to specify the area. You can also change the area with the drop-down menu. The drop-downs will vary depending on the country selected.
  4. Once you've made your selections, click Next.

    The results appear on the map.

  5. Click Done.

    Now that you've saved your site, you can create standard reports and comparison reports using Get Reports and Organize your sites.
