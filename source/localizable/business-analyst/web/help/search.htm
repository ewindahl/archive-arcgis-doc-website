Search

The Search tool provides you with options for searching for a location.

Search can be accessed in the Maps tab.

  • You can enter an address, place or city and press Enter.
  • Using the tool drop-down menu, you can add a pin, add a boundary, zoom to an area or perform a Business and Facilities search or Web Maps search.

If you enter information and click Enter, a pin is placed on the map by default. If you use Zoom to an area, a pin is not placed on the map.

Add a pin

You can add a pin to the map to indicate a location. Once you've added the pin, you can add rings, drive time and walk time; move the point; and delete the point.

  1. In the Maps tab, click the arrow in the Search tool.
  2. Click Add a pin.
  3. Enter an address, place or city and press Enter.

Add a boundary

You can use Add a boundary to create boundaries on the map for ZIP Codes, cities, counties or states.

  1. In the Maps tab, click the arrow in the Search tool.
  2. Click Add a boundary.
  3. Enter a ZIP Code, city, county or state and press Enter.
    4. The boundary appears on the map with options to get reports for the boundary or to remove the boundary.
    Note:

    If you select Get Reports, the boundary is converted to a site and saved in My Sites.

Zoom to an area

You can zoom to an area on the map for a closer look. Some examples of areas include, but aren't limited to, ZIP Codes, cities, counties, states and neighborhoods.

  1. In the Maps tab, click the arrow in the Search tool to access the search options.
  2. Click Zoom to an area.
  3. Enter an area and press Enter.
    4. The map area is zoomed in.

Business and Facilities Search

Business and Facilities Search is accessible from the Search tool or from the Create Maps from Data tab.

Note:

Business and Facilities Search is not available for all countries.

  1. In the Maps tab, click the arrow in the Search tool to access the search options.
  2. Click Business and Facilities Search.
  3. Enter the business or facility name or type in the text box and press Enter.
    4. The Business and Facilities Search dialog box appears. See Business and Facilities Search for more information on refining your search, editing the symbol, choosing a style and saving your search results.

Web Maps Search

Web Maps Search is accessible from the Search tool or from the search option in Web Maps via More Maps.

  1. In the Maps tab, click the arrow in the Search tool to access the search options.
  2. Click Web Maps Search.
  3. Enter the web map or group in the text box and press Enter.
    4. The Browse ArcGIS Online Web Maps dialog box appears with your search results.
