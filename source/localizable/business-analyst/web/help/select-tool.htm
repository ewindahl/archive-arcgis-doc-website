Select tool

The Select tool can be accessed from the tool palette and is combined with the Pan tool. You can select features on the map such as polygons, rings, drive time, walk time, heat maps, color-coded maps, Smart Map search results, or features on a web map. You can also combine these features to create one site and create a report.

Use the Select tool for polygons

You can select a polygon and create reports.

With a polygon on the map, click the polygon or press Shift and drag the pointer to select it. A pop-up appears with options to get reports.

Combine polygons

You can combine polygons to create one site and then create reports for the combined site.

  1. With more than one polygon on the map, press Shift and drag the pointer over the polygons to select them.

    The polygons are shaded to indicate they are selected.

  2. In the dialog box allowing you to combine the sites, click Yes.
  3. A dialog box appears with the combined site and a link to create a report. Click Get reports.

Use the Select tool for web maps

You can select a feature on a web map and create reports.

With a web map feature on the map, click it or press Shift and drag the pointer to select it. A pop-up appears with options to get reports.

Combine features on a web map

You can combine features on a web map to create one site and then create reports for the combined site.

  1. With multiple features on a web map, using the Select tool, press Shift and drag the pointer over the features to select them.
  2. A dialog box appears indicating the number of features selected. You can scroll through them using the arrows. Click Yes to combine the selected sites into one site.
  3. A dialog appears with the combined site and a link to create a report. Click Get reports.

Use the Select tool for points

You can select a point on a the map and create reports.

With a point on the map, click on it or press Shift and drag the pointer to select it. A pop-up appears with options to get reports.

Combine points

You can combine points to create one site and then create reports for the combined site.

  1. With multiple points on the map, using the Select tool, press Shift and drag over the points to select them.
  2. In the following dialog box, click rings, drive time and walk time and click Apply.
    3. The rings, drive time or walk time appear on the map.

Use the Select tool for heat maps

You can select individual and multiple polygons on a heat map. You can also select the underlying point data. Heat maps styles can be created for maps when using Business and Facilities Search and Import File for importing an Excel file or a point shapefile.

With the heat map style selected on the map, click the Pan and select tool from the tool palette.

  • To select one polygon, click on it. A pop-up appears to get reports.
  • To select individual polygons, press Ctrl and click on the polygons. A pop-up appears to get reports.
  • To select the underlying point data, press Shift and drag over the area. A pop-up appears to create rings, drive time and walk time.

Use the Select tool in Smart Map Search and Color-Coded Maps

You can select an area shown on the map and create a report.

Once you've selected variables in Smart Map Search or Color-Coded Maps and they are shown on the map, click the area or press Shift and drag the pointer to select it. A pop-up appears with options to get reports.

Combine areas

You can combine areas to create one site and then create reports for the combined site.

  1. While viewing the map of the selected variables, press Shift and click or drag the areas to select them.

    For example, if you have selected a variable for counties, you can press Shift and select the counties to create reports.

  2. A dialog box appears indicating the number of polygons selected. You can scroll through them using the arrows. Click Yes to combine the selected sites into one site.
  3. A dialog box appears with the combined site and allows you to instantly create a report.
